# AI-Doctor Backend API

## 🚀 Başlangıç

### Kurulum
```bash
npm install
```

### Geliştirme Modu
```bash
npm run dev
```

### Production
```bash
npm start
```

## 📡 API Endpoints

- `GET /health` - Sunucu sağlık kontrolü
- `GET /api` - API versiyon bilgisi

## 🔧 Ortam Değişkenleri

`.env.example` dosyasını `.env` olarak kopyalayın ve kendi değerlerinizi girin.
